This script is used for sequence motif analyses. Running the python file named 'RunMotifAnalysis' will ask you to first choose one reference fasta file (which can be input library data), and then you can choose multiple fasta files to be analyzed compared to the reference file.

It is recommended to include in one folder all the Motif-Analysis scripts, as well as the fasta files that would be used for the analysis. All output files would be generated within that folder.
